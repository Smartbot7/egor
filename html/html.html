HTML — язык гипертекстовой разметки.

Он нужен для для структурирования и отображения контента на веб странице. Для HTML суще ствуют различные команды:

1) Самыми частыми командами для вас будут "div" и "p". Они нужны для создания "контейнеров" для классификации будущих обьектов через стили в CSS

2) Для придания размера тексту без использования CSS используют "h". Например h1 h3 h5 вплоть до h6
