HTML — язык гипертекстовой разметки.

Он нужен для для структурирования и отображения контента на веб странице. HTML использует различные команды:

1) Самыми частыми командами для вас будут "div" и "p". Они нужны для создания "контейнеров" и классификации будущих обьектов через стили в CSS

2) Для придания размера тексту без использования CSS используют "h". Например h1, h3, h5 вплоть до h6

Это h1

Это h3

Это h5

Как видно с большим числом уменьшается и текст (Кстати этот проект написан на h2)

3) С помощью значка "!" в начале нового проекта можно создать "Базовую разметку". Она состоит из: "html" "head" и "body"

В теге "html" состоят асолютно все другие теги необходимые для работы

Тег "head" предназначен для хранения других элементов, цель которых — помочь браузеру в работе с данными (Также внутри контейнера "head" находятся метатеги, которые используются для хранения информации предназначенной для браузеров и поисковых систем.). Например тег "title" определяет заголовок документа, а тег "meta" определяет те самые метатеги
